# Mini Projet : Weather App

C'est une application web simple pour voir la météo d'une ville en temps réel. J'ai réalisé ce projet pour pratiquer HTML, CSS et JavaScript avec l'API OpenWeather.

# Fonctionnalités
 **Recherche :** On peut écrire le nom d'une ville pour voir sa météo.
 **Affichage :** L'application affiche la température, l'humidité et une icône.
 **Erreur :** Si la ville n'existe pas, un message d'alerte s'affiche.
 **Dark Mode :** Un bouton pour changer entre le mode jour et le mode nuit.

# Technologies
 **HTML & CSS** (Responsive design).
 **JavaScript** (Utilisation de Fetch API).

# Comment lancer le projet
1. Téléchargez les fichiers du projet.
2. Ouvrez le fichier `index.html` dans votre navigateur (Chrome, Edge, etc.).

Note : La clé API est déjà incluse dans le fichier script.js.
